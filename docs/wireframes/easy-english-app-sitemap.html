핵심 흐름 요약

🆕첫 방문 흐름

  1. 홈 진입 → localStorage 없음 → Empty 상태
  2. "알파벳 시작하기" → /alphabet 목록
  3. A 선택 → /alphabet/A 학습 세션
  4. Step 1: 카드 보기 + 자동 TTS → "다음"
  5. Step 2: 퀴즈 (index%3 유형, index%2 방향)
  6. 정답 시 TTS 자동 재생 → 다음 항목(B) 이동
  7. 마지막 항목 완료 → 복습 or 문장 연습 단계
  8. 문장 연습 3문제 → 완료 화면 → 스트릭 업데이트

📚단어 학습 (퀴즈 순환)

  1. 홈 → 단어 카드 → /words
  2. 단어 선택(apple, index=0) → 세션 진입
  3. index%3=0 → ImageChoiceQuiz, index%2=0 → en→ko
  4. dog(1): MatchingQuiz + ko→en 방향
  5. bird(2): ListenChoiceQuiz + en→ko 방향
  6. fish(3): ImageChoiceQuiz + ko→en (3번째 순환)
  7. 각 완료 → localStorage 저장 → 다음 진행

🔁오답 복습 라운드

  1. 퀴즈 틀린 항목 → wrongItems[]에 추가
  2. 마지막 항목 Step 2 완료 → 개수 체크
  3. 오답 있음 → 복습 라운드 (헤더 주황)
  4. 오답 없음 → 문장 연습 단계 바로 이동
  5. 복습: 유형 A 고정, en→ko, 1회만
  6. 또 틀려도 정답 확인 후 진행 (재복습 없음)
  7. 복습 마지막 완료 → 문장 연습 단계

📝문장 연습 단계

  1. 복습 완료 후 (or 오답 없으면 바로) 진입
  2. SENTENCE_COUNT=3 문제 순서대로 진행
  3. sentences.ts에서 sentenceIndex % 20 문장 선택
  4. 타일 클릭 or 직접 입력으로 한글 문장 조합
  5. 정답: onCorrect → 다음 문장 또는 완료
  6. 3문제 완료 → updateStreak() → /complete

🔥스트릭 업데이트 로직

  1. 앱 로드 시 lastStudiedDate 확인
  2. 어제 학습했으면 streak + 1
  3. 이틀 이상 공백 → streak 0 리셋
  4. 문장 연습 완료 시 updateStreak() 호출
  5. 완료 화면에서 업데이트된 streak 표시

🔡퀴즈 방향 & 유형 규칙

  1. type = ['image','matching','listen'][index % 3]
  2. direction = index % 2 === 0 ? 'en→ko' : 'ko→en'
  3. index 0 → 그림선택 en→ko
  4. index 1 → 매칭 ko→en
  5. index 2 → 소리선택 en→ko
  6. index 3 → 그림선택 ko→en (2번째 순환)

🔒회화 잠금 해제

  1. 홈 회화 카드 → wordProgress < 50이면 잠금
  2. 단어 50개 완료 시 isPhraseUnlocked() = true
  3. 회화 카드 잠금 해제 → /phrases 접근
  4. (Phase 2 구현, MVP는 잠금 상태 유지)

⚠️오류 처리

  1. window.speechSynthesis 미지원 감지
  2. supported=false → 자동 TTS 스킵
  3. 발음 없이 학습 계속 진행 가능
  4. localStorage 쓰기 실패 → 세션 메모리 폴백
