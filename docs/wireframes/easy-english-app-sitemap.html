핵심 흐름 요약

🆕 첫 방문 흐름

  1. 홈 진입 → localStorage 없음 → 첫 방문 Empty 상태 표시
  2. "알파벳 시작하기" → /alphabet 목록
  3. A 선택 → /alphabet/A 학습 세션
  4. Step 1 카드 보기 → 발음 듣기 → "다음"
  5. Step 2 유형 A 퀴즈 (wordIndex%3=0 → image-choice, wordIndex%2=0 → en→ko)
  6. Step 3 발음 확인 → "완료" → A 진도 저장
  7. 다음 항목(B) 자동 이동 또는 오늘 세션 종료 → 완료 화면
  8. 완료 화면 → "홈으로" → 스트릭 업데이트 확인

📚 단어 학습 흐름 (퀴즈 순환)

  1. 홈 → 알파벳 완료 → 단어 카드 클릭 → /words
  2. 단어 선택(예: apple, index=0) → 학습 세션
  3. Step 1 FlashCard: apple / 🍎 / 사과 / 발음
  4. Step 2: index%3=0 → ImageChoiceQuiz, index%2=0 → en→ko 방향
  5. dog(index=1): MatchingQuiz + ko→en 방향
  6. bird(index=2): ListenChoiceQuiz + en→ko 방향
  7. fish(index=3): ImageChoiceQuiz + ko→en 방향 (순환)
  8. 각 완료 → localStorage 저장 → 다음 단어 자동 진행

🔁 오답 복습 라운드 흐름

  1. 세션 진행 중 퀴즈에서 틀린 항목 → wrongItems[]에 추가
  2. 세션 마지막 항목 Step 3 완료 → wrongItems.length > 0 체크
  3. 오답 있음 → 복습 라운드 진입 (헤더 주황색으로 변경)
  4. 오답 없음 → 바로 완료 화면으로 이동
  5. 복습: 유형 A(그림 선택) 고정, en→ko 방향, 1회만
  6. 복습에서 또 틀려도 정답 확인 후 다음 진행 (재복습 없음)
  7. 복습 마지막 항목 완료 → 완료 화면

🔥 스트릭 업데이트 로직

  1. 앱 로드 시 lastStudiedDate 확인
  2. 오늘 날짜와 비교 → 어제 학습했으면 streak+1
  3. 이틀 이상 공백 → streak 0으로 리셋
  4. 학습 완료 후 lastStudiedDate = 오늘로 갱신
  5. 완료 화면에서 업데이트된 streak 표시

🔒 회화 잠금 해제 조건

  1. 홈에서 회화 카드 → wordProgress.length < 50이면 잠금 표시
  2. 단어 50개 완료 시 → isPhraseUnlocked() = true
  3. 회화 카드 잠금 해제 → /phrases 접근 가능
  4. (Phase 2 구현 시 활성화, MVP에선 잠금 상태 유지)

⚠️ 오류 처리

  1. Web Speech API window.speechSynthesis 미지원 감지
  2. useSpeech.ts: supported=false → speak() 호출 시 안내 모달
  3. 학습은 발음 없이 계속 진행 가능 (버튼 비활성화)
  4. localStorage 쓰기 실패 → catch → 세션 메모리로 폴백

🔡 퀴즈 방향 & 유형 규칙

  1. type = ['image-choice','matching','listen-choice','sentence-builder'][index % 4]
  2. direction = index % 2 === 0 ? 'en-to-ko' : 'ko-to-en'
  3. index 0 → 그림선택 + en→ko / index 1 → 매칭 + ko→en
  4. index 2 → 소리선택 + en→ko / index 3 → 문장조합 + ko→en
  5. 문장조합: 영어 목표 문장 표시 → 한글 타일 탭하여 순서대로 조합
  6. 틀려도 정답 확인 후 진행 (막지 않음)
