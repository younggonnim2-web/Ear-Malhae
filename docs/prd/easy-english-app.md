## 한 줄 정의

영어 초보자(초등 저학년·노년층)가 **하루 5분, 부담 없이** 단어를 익히고 회화까지 나아갈 수 있는 초간단 영어 학습 앱.

---

## 문제 정의

| # | 질문 | 답변 요약 | 인사이트 |
|---|------|----------|---------|
| 1 | 수요 | 초급인데도 어렵게 느껴져 매일 학습 지속이 안 됨 | **"어렵다는 느낌"이 최대 장벽** — 난이도가 아니라 인지적 부담이 문제 |
| 2 | 현상 유지 | 학원·유튜브는 별도 시간 필요 → 지속 불가 | 일상 틈새에 끼워 넣을 수 있는 **마이크로 러닝** 형태 필요 |
| 3 | 구체성 | 단어 암기 + 회화 연습 두 가지 | 단계적 접근: 단어 → 문장 → 회화 |
| 4 | MVP | 단어 암기 | 가장 작은 검증 단위 확인됨 |
| 5 | 관찰 | 1년 학원 수강 후에도 알파벳 미완성 | 기존 방식은 **초보자에게 실질적으로 작동 안 함** — 진짜 초급 설계 필요 |
| 6 | 미래 | AI 회화 연습 + 발음·문법 교정 | AI 피드백이 핵심 차별점 |

## 타깃 사용자

### 페르소나 A — 초등 저학년 (6~9세)
- 알파벳 미완성 수준
- 집중 시간 5분 이하
- 시각적·청각적 자극에 반응
- 부모가 옆에서 같이 쓸 가능성 높음

### 페르소나 B — 노년층 (60세 이상)
- 스마트폰 사용 미숙
- 글씨 크고 단순한 UI 필수
- 반복 학습 선호
- "틀리면 창피하다"는 심리적 부담 큼

## 핵심 문제 (Why Now)

- 기존 앱(Duolingo 등)도 초보자에게 여전히 **인지 부담이 높음**
- 학원은 **시간·비용·이동** 세 가지 장벽
- AI 음성 인식·교정 기술이 성숙 → 저비용 회화 피드백 가능해진 시점

## 기능 범위

### Phase 1 — MVP (단어 암기)
- [ ] 알파벳 26자 학습 (그림 + 소리)
- [ ] 기초 단어 카드 (100개, 그림·발음 제공)
- [ ] 퀴즈 4가지 유형 (단어별 순환 배정)
  - 그림 보고 선택 (4지선다)
  - 단어 매칭 (영어↔한글 짝 맞추기)
  - 소리 듣고 선택
  - 문장 조합 (배운 단어로 짧은 문장 만들기 — 타일 선택)
- [ ] 퀴즈 방향 교차 노출 (영어→한글 / 한글→영어 번갈아)
- [ ] 틀려도 격려 메시지, 정답 확인 후 진행
- [ ] 세션 완료 후 오답 복습 라운드 (틀린 항목만 퀴즈 재시도, 1회)
- [ ] 하루 학습 스트릭(streak) 표시
- [ ] 큰 글씨, 단순 UI (최소 버튼 수)

### Phase 2 — 문장·회화 (추후)
- [ ] 기초 회화 문장 학습 (AI 발음 교정 포함)
- [ ] AI 음성 인식으로 발음 교정
- [ ] 문법 오류 피드백
- [ ] 학부모/보호자 학습 리포트

## 비 범위 (Out of Scope)

- 문법 설명 (초급에선 불필요)
- 복잡한 레벨 시스템
- 소셜/랭킹 기능 (심리적 부담 유발)
